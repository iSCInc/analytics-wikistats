    WiViVi = Wikipedia Views Visualized

Map Wikipedia pageviews, percentage to language ... shows the relative 'popularity' of any particular Wikipedia language version in different countries.

Info panels reveal details about the selected language (button 'Breakdown ..') or any country (hover over it).

Metrics include population size, percentage connected to internet, monthly views, etc.

A second map Pageviews to any Wikipedia (button top-left) focuses on in which countries Wikipedia is read most, regardless of language.

Best viewed on a desktop monitor or tablet. On small mobile screens some panels can not be shown. skip next time
